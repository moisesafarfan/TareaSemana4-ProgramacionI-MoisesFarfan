Breve explicación del programa: 

Este programa solicita al usuario su nombre completo, el precio de un producto y la cantidad comprada.
Calcula el subtotal multiplicando la cantidad por el precio, luego calcula el IVA como el 12% del subtotal.
El total a pagar se obtiene sumando el subtotal y el IVA.
Se utiliza getline() para capturar nombres con espacios y cin para los valores numéricos.
Se aplica fixed y setprecision(2) para mostrar los valores monetarios con dos decimales.
Finalmente, imprime en pantalla el nombre del cliente, subtotal, IVA, total, se aplica un salto de linea y  el nombre del estudiante que desarrolló el programa.
